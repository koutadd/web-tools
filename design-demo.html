DESIGN.md Demo — Claude Style

AIエージェントがそのブランドのUIを作る

getdesign.md を導入すると、AIコーディングエージェントが選んだブランドのデザインシステムに沿ってUIを実装します。

getdesign.md を開く ブランド一覧を見る

どうやって動くのか

3ステップで、AIがブランドのスタイルでUIを生成します。

Step 1

ブランドを選ぶ

66ブランドのDESIGN.mdから使いたいスタイルを選択します。Vercel・Stripe・Linear・Claude など。

Step 2

npxで追加する

npx getdesign@latest add stripe を実行するだけ。プロジェクトにDESIGN.mdが生成されます。

Step 3

エージェントに伝える

Claude CodeやCursorがDESIGN.mdを読み、そのブランドのカラー・タイポ・コンポーネント仕様でコードを書きます。

現在のDESIGN.md: Claude Style

このページ自体がClaude Anthropicスタイルで実装されたデモです。Parchment背景・Terracottaアクセント・Georgiaセリフ体。

カラー

Parchment + Terracotta

羊皮紙色のベース (#f5f4ed) にテラコッタアクセント (#c96442)。クールな青みは一切使わない。

タイポ

Serif for 見出し

Anthropic Serif (Georgia代替) をweight 500で統一。セリフ体が本のタイトルのような格調を演出。

シャドウ

Ring Shadow システム

0px 0px 0px 1px のリングシャドウ。ドロップシャドウではなくボーダーのように機能する独自の深さ表現。

ブランドを切り替える

下のブランドを選んで npx コマンドを実行すると DESIGN.md が切り替わります。

claude vercel linear cursor stripe spotify supabase raycast figma notion

▸ ターミナルで実行 (--force で上書き)

npx getdesign@latest add claude --force
